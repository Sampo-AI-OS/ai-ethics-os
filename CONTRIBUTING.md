# Contributing

Thanks for taking the time to look at AI Ethics OS.

This repository is a curated public edition in the Sampo AI OS portfolio. The intent is to keep it approachable, technically credible, and collaboration-friendly.

## Good Contribution Targets

Useful contributions include:

- documentation clarity improvements
- bug fixes in the public edition
- developer-experience improvements
- frontend polish in the standalone demo
- tests for the public scoring workflow
- issue reports with clear reproduction steps

## Before You Contribute

Please keep these constraints in mind:

- this repository is a public showcase edition, not the complete internal product
- some commercially sensitive components are intentionally omitted
- contributions should fit the public-edition scope instead of trying to reconstruct the removed private core

## Preferred Workflow

1. Open an issue for larger changes so the direction can be discussed first.
2. Keep pull requests focused and easy to review.
3. Update documentation when behavior changes.
4. Add or update tests when public behavior changes.

## Tone And Collaboration

The goal of this repository is to be useful, respectful, and welcoming.

Constructive feedback, thoughtful criticism, and practical collaboration are all welcome.
